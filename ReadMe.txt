	Olá, essa é a versão 1.0.0 alpha do software AutoHouse.Pensando na evolução da Internet Of Things (IoT), criaram essa
interface gráfica os alunos de engenharia da computação: Gustavo Damasceno (programação, design e visual), Lennon Ribeiro
(programação, desingn, visual  e lógica estrutural) e Moisés Queiroz (programação e lógica estrutural) primeiramente focada na primeira avaliação da matéria Estrutura de Dados, porém, visando a longo prazo utilizar a mesma para controle residencial com o auxilio de microcontroladores e/ou microcomputadores. 
	Como ainda é um software em versão de testes, é necessário utilizar uma IDE (NetBeans, Eclipse e etc) para visuali-
zar seu funcionamento ou pelo terminal, para tal ação deve ser executado o arquivo run.java.Caso queira contribuir com o projeto, entre em contato.
